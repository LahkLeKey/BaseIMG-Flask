## Fresh Install

1.  `ssh USERNAME@IP`
2.  `git clone https://github.com/LahkLeKey/BaseIMG-Flask.git`
3.  `cd BaseIMG-Flask`
4.  `docker-compose up -d`
5.  Visit IP:5000 and application should be deployed.
